# status.

Добро пожаловать в **status.** — социальную сеть и иммерсивный симулятор жизни.
Создайте уникального персонажа, исследуйте динамично генерируемые миры, общайтесь с NPC на базе искусственного интеллекта, выполняйте квесты и стройте отношения в живой симуляции!

[English Version](README.md)

## Особенности
- **Динамичные ИИ-миры**: Создайте мир, опишите его лор, и ИИ начнет генерировать события, квесты и уникальных персонажей на основе ваших вводных данных.
- **Живая лента новостей**: Читайте посты от лица "Системы" (например, от Вестника Мира) и ваших NPC. Мир напрямую реагирует на ваши действия и квесты.
- **Отношения и общение**: Общайтесь с персонажами в ветках комментариев под постами. Ваши ответы влияют на шкалу отношений с каждым из них.
- **RPG Прогрессия**: Повышайте уровень, получайте опыт (XP) за квесты и ивенты, и открывайте новые уникальные навыки, сгенерированные ИИ специально для вас.
- **Импорт / Экспорт**: Сохраняйте весь прогресс мира в JSON файл на ваше устройство и импортируйте его обратно в любой момент, чтобы не потерять сохранения или перенести их.

## Технологии
- **Frontend**: React + Vite, Tailwind-like custom CSS
- **Мобильный Контейнер**: Capacitor (для сборок под Android & iOS)
- **Управление состоянием**: Zustand
- **Интеграция ИИ**: Прямая интеграция LLM API (Gemini)

## Установка и запуск
1. Склонируйте репозиторий.
2. Перейдите в папку `app`: `cd app`
3. Установите зависимости: `npm install`
4. Локальный запуск (web): `npm run dev`
5. Сборка под Android: `npm run build && npx cap sync android`
   - Откройте проект в Android Studio или соберите APK напрямую: `cd android && gradlew assembleDebug`.

## Экспорт и Импорт
Все данные миров хранятся исключительно на вашем устройстве. Используйте кнопки `Экспорт` и `Импорт` на главном экране, чтобы сохранять резервные копии миров в папку Документов (Documents).
